# chakra-agent-skills

A collection of specialized skills for Chakra UI v3 development and React component building. These skills provide comprehensive documentation, usage patterns, and best practices to accelerate UI development.

## Skills Overview

### Chakra UI v3 Skills

| Skill | Description | When to Use |
|-------|-------------|-------------|
| [`chakraui-components`](skills/chakraui-components/SKILL.md) | Complete component reference for Chakra UI v3 | When implementing UI components, forms, dialogs, tables, navigation, overlays, or layouts |
| [`chakraui-styling`](skills/chakraui-styling/SKILL.md) | Styling patterns, responsive design, and CSS props | When applying styles, responsive breakpoints, conditional styles, or layout properties |
| [`chakraui-theming`](skills/chakraui-theming/SKILL.md) | Theme customization, tokens, and design system | When customizing themes, creating design tokens, or configuring the Chakra design system |

## Getting Started

### Prerequisites

- Node.js and npm/yarn
- Basic knowledge of React and TypeScript
- Familiarity with Chakra UI concepts

### Installation

Clone the repository:

```bash
git clone <repository-url>
cd mrmo-skills
```

### Usage

These skills are designed to work with AI assistants and coding tools that support skill-based workflows. Each skill contains:

- **Comprehensive documentation** covering all aspects of Chakra UI v3
- **Practical examples** with code snippets
- **Best practices** and common patterns
- **Reference materials** for quick lookup

## Skill Structure

Each skill follows a consistent structure:

```
skill-name/
├── SKILL.md              # Main skill documentation and instructions
└── references/           # Detailed reference materials
    ├── component-docs.md
    ├── examples.md
    └── best-practices.md
```
